# OptiDocs AI

OptiDocs AI is an AI-powered document processing tool that allows users to compress images to specific file sizes, adjust brightness, convert between formats, and smart-scan documents using Gemini AI.

This project was built using React, TypeScript, and Lucide React. It utilizes the Gemini AI API for document processing and is designed to be user-friendly and efficient.

## Features

* Compress images to specific file sizes
* Adjust brightness and contrast of images
* Convert between image formats (e.g. JPEG to PNG)
* Smart-scan documents using Gemini AI

## Technologies Used

* React
* TypeScript
* Lucide React
* Gemini AI API

## Installation and Usage

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm start`
4. Open the application in your web browser at `http://localhost:3000`
